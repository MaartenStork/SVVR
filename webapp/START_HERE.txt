WHAT YOU HAVE:
- A complete React web app with Flask backend

================================================================================
QUICK START - RUN LOCALLY (5 MINUTES)
================================================================================

STEP 1: Install Backend Dependencies
-------------------------------------
cd backend
pip install -r requirements.txt

STEP 2: Install Frontend Dependencies
--------------------------------------
cd ../frontend
npm install

STEP 3: Start Backend Server
-----------------------------
Open Terminal 1:
cd backend
python app.py

(Keep this running)

STEP 4: Start Frontend
-----------------------
Open Terminal 2:
cd frontend
npm start

(Browser will open automatically to http://localhost:3000)

STEP 5: Use the App
-------------------
1. You'll see a welcome screen
2. Enter three hot-square sizes (default: 0.1, 0.2, 0.33)
3. Click "Run Simulations"
4. Watch real-time simulations and convergence graph!

================================================================================
DEPLOY TO RENDER CLOUD (10 MINUTES)

STEP 2: Deploy on Render
-------------------------
1. Go to: https://dashboard.render.com/
2. Click: "New" -> "Blueprint"
3. Connect your GitHub repository
4. Select the webapp directory
5. Click "Apply"
6. Wait 5-10 minutes
7. Done! Your app is live!

================================================================================
WHAT IT DOES
================================================================================

The app runs your Jacobi heat simulation (from code.py) in real-time and shows:
- Three simulations side-by-side with different hot-square sizes
- Live temperature visualizations with turbo colormap
- Real-time convergence graph comparing all three
- Answers: "Does hot-square size affect solve difficulty?"
  Answer: YES! Larger squares converge FASTER!

================================================================================
TROUBLESHOOTING
================================================================================

Backend won't start?
  -> Check Python version: python --version (need 3.9+)
  -> Reinstall: pip install -r requirements.txt

Frontend won't start?
  -> Delete node_modules and reinstall: rm -rf node_modules && npm install

Can't connect?
  -> Make sure backend is running on port 5000
  -> Check browser console for errors
